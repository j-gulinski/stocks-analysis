# Stock Analysis Workbench

`AGENTS.md` is the single authoritative project instruction file. Read it
before changing code, schema, plans, configuration, or scraper behavior.

Use the on-demand document index in `AGENTS.md`; do not duplicate its rules in
this file. In particular:

- current status and execution order live in `TASKS.md`;
- architecture and the RT roadmap live in `docs/plan-research-platform.md`;
- product and quality gates live in `docs/project-guardrails.md`;
- strategy behavior lives in `skill/SKILL.md` and `docs/strategy-malik.md`;
- scraper quirks live in `skills/scraper-doctor/SKILL.md`.

This compatibility file exists for tools that still look for `CLAUDE.md`.
