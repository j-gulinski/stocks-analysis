# Worked examples (few-shot grounding)

Three real cases distilled from Malik's own reasoning in `obs.txt` / the OBS
forum portfolio. They are teaching examples for the analyst prompt — they show
the *shape* of a good verdict, including how to handle the strategy's data gaps
and, crucially, one **documented miss** so the model learns the failure mode,
not just the wins.

- `optex.md` — the textbook pattern: cheap-on-forward-earnings + a quantified,
  not-yet-priced catalyst (backlog) + margin of safety, and sell discipline when
  the catalyst stops growing. **Outcome: +100% in ~6 months.**
- `toya.md` — the durable "always-cheap for a structural reason" case, where a
  buyback is treated as an active catalyst underwriting the valuation gap.
  **Outcome: multi-year, repeatedly re-confirmed discount.**
- `suntech.md` — the miss: a narrative-catalyst thesis (new contracts) that
  repeatedly failed to materialise, held anyway by reclassifying it as "a hold
  for years". **Outcome: Malik's own confessed rationalisation trap.**

These are illustrative summaries, not live dossiers; numbers are as Malik cited
them at the time. Prefer the actual dossier figures when analysing a real
company.
